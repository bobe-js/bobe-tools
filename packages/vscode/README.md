# Bobe

VS Code language support for Bobe templates.

## Features

- Syntax highlighting for `.bobe` files.
- Inline Bobe template highlighting in JavaScript, TypeScript, JSX, TSX, Vue, Svelte and HTML-like files.
- TypeScript server plugin support for Bobe template diagnostics and language features.

## Requirements

No additional setup is required.
