
## API

```typescript
class Parent {
	ui = bobe`
		${Child} header=
      div text="header头"
    | 
      div	text="children"
	`
}


class Child {
	ui = bobe`
		div
			div id="header"
				{header}
		div
    	div id="children"
    		{children}
	`
}
```

Compiler 解析支持

1. 基于  `parsePropertyInlineFragment` 生成
   1. 用 `componentNode.inlineRenderNames` 来保存 `Map<propKey, inlineRenderName>`

2. `parseComponentNode - propsAdded - createSetPropsExp` 钩子
   1. 如果 key 存在于 Map 中 就使用 ${nameDot}${key}=inlineRenderName` 渲染
   2. 否则 使用常规赋值
