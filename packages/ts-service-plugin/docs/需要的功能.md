**语法高亮 (Syntax Highlighting)**

- 为 bobe 模板提供高亮语法

**智能补全与感知 (IntelliSense)**

- 支持 JS/TS 补全，dom 标签，属性补全

**语义类型检查 (Type Checking)**

- 当属性值 和 属性名的类型不匹配时触发

**代码导航 (Navigation)**

- 支持“转到定义”（Go to Definition）、“查找引用”（Find References）。例如，在模板里点击一个自定义组件，可以跳转对应 ts 代码位置

**重构支持 (Refactoring)**

- 支持跨文件的重命名（Rename）。重构一个组件的 Prop 名称，插件会自动更新所有引用该组件的模板代码。
